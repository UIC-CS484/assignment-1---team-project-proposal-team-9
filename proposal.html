Cryptocurrency Portfolio Monitor

APIs we will use:
Nomics API for cryptocurrency market information
FTX API for futures and account information
Exchange API for user account information (Coinbase, Binance, etc.)

The data recieved from the APIs will allow the user to monitor their cryptocurrency positions across multiple different exchanges, as well as aggregate and manipulate information from multiple accounts.

From authenticated users we will collect email address, password, any manually entered positions, and private API keys/secrets to access users' exchange account information.

USER -> fname, lname, email, pkey
COIN -> symbol, qty, price, entry

This site could be used by retail cryptocurrency investors. The goals of the site are to provide a convenient location for users to monitor their positions across multiple accounts to have an overview of all of their investments, and create custom price tracking alerts.
